# QuadSync
This is the GitHub repository for the code for the CVPR 2026 paper "QuadSync: Quadrifocal Tensor Synchronization via Tucker Factorization" by Daniel Miao, Gilad Lerman, and Joe Kileel. 

Check the load_image_ETH3D and joint_load_image_ETH3D for the full pipeline that was used to run the quad_sync and joint_sync algorithms. 

Contact miao0022@umn.edu for questions or concerns.
